# Análise de Documentos Anti-Fraude com Azure AI

Este projeto foi desenvolvido no Bootcamp da DIO e utiliza a Azure AI para analisar documentos de forma automática, identificando possíveis fraudes.

Funcionalidades:
- Upload de documentos para o Azure Blob Storage
- Análise dos documentos usando o Azure Form Recognizer
- Validação se o documento é válido
- Exibição do arquivo e resultados na tela

Para rodar o projeto, configure as chaves da Azure e siga as instruções no código.
